Corrections possibles

1/ Compter le nombre de Thriller présents en base  (films qui possèdent au moins le genre Thriller)
db.getCollection('mflix_movies').count({ genres: "Thriller" }) db.getCollection('mflix_movies').count({genres: { $all: ["Thriller"] }}) db.getCollection('mflix_movies').count({genres:{ $elemMatch: { $eq: "Thriller" }}})   OU un .find(avec votre requête).count()
2/ Trouver la liste des films qui contiennent le mot "ghost" dans leur titre 
db.getCollection('mflix_movies').find({ title: /ghost/i })
3/ Trouver la liste des films qui contiennent le mot "ghost" dans leur titre et qui sont sortis après 2013 
db.getCollection('mflix_movies').find({ title: /ghost/i, year: {$gte: 2013 }})
4/ Trouver le film qui a gagné le plus de récompenses 
db.getCollection('mflix_movies').aggregate([  
   {$sort : {"awards.wins" : -1}},  
   {$limit : 1 } 
])
5/ Trouver le plus vieux film de plus de 2 heures ayant une note inférieur à 2 sur la plateforme imdb 
db.getCollection('mflix_movies').aggregate([ 
      {$match: { runtime: {$gte: 120}, "imdb.rating": {$lte: 2.0} }},
     {$sort : {"released" : 1}},
     {$limit : 1 }
  ])
6/ Modifier la requête précédente pour récupérer en même temps les commentaires. 
db.getCollection('mflix_movies').aggregate([
       {$match: { runtime: {$gte: 120}, "imdb.rating": {$lte: 2.0} }},
      {$sort : {"released" : 1}},
      {$limit : 1 },
     {$lookup: { from: "mflix_comments", localField: "_id", foreignField: "movie_id", as: "comments" }}
])

  Ce qui est très important ici c'est de ne faire le lookup qu'à la toute fin de l'agrégation, sinon on va effectuer la jointure sur plusieurs documents source alors que seul le plus vieux nous intéresse.

7/ Récupérer le nombre de films par pays ainsi que leurs titres.
Le résultat devra être trié du pays avec le plus grand nombre de films au plus petit. 
  db.getCollection('mflix_movies').aggregate([
      {$unwind: "$countries"},
      {$group: { _id: "$countries", titles: {$push: "$title"}, count: { $sum: 1 } }},
     {$sort : {"count" : -1}}
])    
OU    
db.getCollection("mflix_movies").aggregate([
  {$unwind : "$countries"},
  {$group:{     "_id":"$countries",     "movies": {$push:"$title"} }},
 {$project : {"count":{$size : "$movies"}, movies : 1}},
 {$sort : {"count": -1}}
])
